# Superstore Sales Analytics Pipeline

This project simulates a modern analytics engineering workflow using Python, SQL, and Tableau.

## Project Overview

Raw retail sales data is ingested using Python, transformed with SQL into analytics-ready datasets, validated for data quality, and visualized in Tableau dashboards to analyze business performance and profitability.

## Tools Used

- Python (Pandas)
- SQL
- SQLite
- Tableau

## Pipeline Workflow

1. Raw CSV sales data is loaded into a SQLite database.
2. SQL transformations create cleaned and aggregated analytics tables.
3. Data quality checks validate sales and profit metrics.
4. Clean datasets are exported for visualization.
5. Tableau dashboards analyze regional performance, product categories, and discount impacts on profitability.

## Dashboards

1. Retail Sales Performance & Profitability Dashboard
2. Profitability & Discount Impact Analysis

## Key Insights

- Technology products drive the highest revenue.
- High discounts often correlate with lower profitability.
- A small number of products contribute significantly to total sales.
- Regional performance varies across product categories.
